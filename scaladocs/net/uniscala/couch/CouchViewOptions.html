A set of options used for view requests. The default set of options is the same as the defaults in the specification.

Different options may be obtained by chaining the methods named after the corresponding option in the specification, like this: 

  val options = CouchViewOptions().limit(100).groupLevel(3)
An option can also be set back to its default using the similarly named methods, but prefixed by 'default': 
  val options2 = options.defaultLimit

See http://wiki.apache.org/couchdb/HTTP_view_API for a description of the list of options.
